Fue creado por Guido van Rossum y lanzado por primera vez en 1991. Python es conocido por su sintaxis clara y legible, lo que lo hace ideal para principiantes y para el desarrollo rápido de aplicaciones. Algunas características clave de Python incluyen: Interpreted: Python es un lenguaje interpretado, lo que significa que el código se ejecuta línea por línea. High-level: Python es un lenguaje de alto nivel, lo que significa que abstrae muchos detalles del hardware del ordenador. Dynamic Typing: Python utiliza tipado dinámico, lo que significa que no es necesario declarar el tipo de una variable. Extensive Libraries: Python tiene una amplia colección de bibliotecas estándar y de terceros. Las variables en Python se utilizan para almacenar datos. No es necesario declarar el tipo de variable, ya que Python lo infiere automáticamente. x = 5 y = "Hola, Mundo!" Python soporta varios tipos de datos básicos como enteros, flotantes, cadenas de texto, listas, tuplas, diccionarios, etc. entero = 10 flotante = 10.5 cadena = "Python" lista = [1, 2, 3] tupla = (1, 2, 3) diccionario = {"clave": "valor"} Las estructuras condicionales permiten ejecutar diferentes bloques de código en función de ciertas condiciones. if x > 0: print("x es positivo") elif x == 0: print("x es cero") else: print("x es negativo") Los bucles permiten ejecutar un bloque de código repetidamente. Python soporta bucles `for` y `while`. for i in range(5): print(i) while x > 0: print(x) x -= 1 Las funciones en Python se definen utilizando la palabra clave `def` seguida del nombre de la función y paréntesis. def saludar(nombre): print(f"Hola, {nombre}!") Para llamar a una función, simplemente se utiliza su nombre seguido de paréntesis. saludar("Mundo") Un módulo en Python es un archivo que contiene definiciones y sentencias de Python.
